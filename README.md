## 🎨 디자인 컨셉: "Dreamy Water Garden"

물방울이 맺힌 파스텔 정원, 토끼가 뛰노는 환상적인 공간.
핑크와 민트의 조화로운 그라디언트, 부드럽게 흐르는 물결 효과.

### 주요 특징

- 🐰 **토끼 귀 모티브**: 네비게이션과 버튼에 토끼 귀 디자인
- 💧 **물방울 애니메이션**: 실시간으로 떨어지는 물방울 배경
- 🎨 **글래스모피즘**: 투명하고 부드러운 카드 디자인
- 🌈 **파스텔 그라디언트**: 핑크, 민트, 라벤더의 조화
- ✨ **마이크로 인터랙션**: 부드러운 호버 효과와 애니메이션

## 📁 페이지 구성

### 1. 홈 (/)
- 쿠아 프로필 및 소개
- 기본 정보, 취향, 통계
- 열혈팬/구독팬 랭킹
- 노래 커버 목록

### 2. 패션방 (/fashion)
- **시즌별 타임라인**: 2024 봄(데뷔) ~ 2025 겨울
- **방셀 (특별 의상)**: 게임, 무대, 컨텐츠별 의상
- 카테고리 필터: 전체, 시즌, 이벤트, 게임, 공연, 일상

### 3. VOD/캐치 (/vod)
- 다시보기 & 캐치클립 분리 뷰
- **컨텐츠별 분류**: 노래, 애교, 게임, 댄스, 소통, ASMR
- 시즌별 통계
- SOOP VOD 플레이어 임베드

### 4. 히스토리 (/history)
- 주요 활동 타임라인
- 구쿠아/뉴쿠아 활동 기간
- 이벤트 및 성과 기록

## 🛠️ 기술 스택

- **React 18** + **Vite** - 빠른 개발 환경
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **Framer Motion** - 부드러운 애니메이션
- **React Router** - 페이지 라우팅

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속

### 빌드

```bash
npm run build
```
